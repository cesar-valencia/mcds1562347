1. Anidar elementos ( > )

Ejemplo: div>div>form>input
<div> <div> <form action=""><input type="text"></form> </div> </div>

2. Elementos hermanos ( + )

Ejemplo: header+nav+article+footer
<header></header> <nav></nav> <article></article> <footer></footer>

3. Repetir elementos ( * )

Ejemplo: form>table>input*5
<form action=""> <table> <input type="text"> <input type="text"> <input type="text"> <input type="text"> <input type="text"> </table> </form>

4. Atributo identificador ( # )

Ejemplo: table#estudiantes
<table id="estudiantes"></table>

5. Atributo clase ( . )

Ejemplo: button.btn
<button class="btn"></button>

5.1 multiples clases ( ... )

Ejemplo: button.btn.btn-primary
<button class="btn btn-primary"></button>

6. Adicionar texto ( { } )

Ejemplo: p{ texto en el parrafo}
<p> texto en el parrafo</p>

7. Personalizar atributos ( [ ] )

Ejemplo: input[type="button"]
<input type="button">

8. Números consecutivos ( $ )

Ejemplo: div.$*3
<div class="1"></div> <div class="2"></div> <div class="3"></div>

8.1 Direccion números consecutivos ( @- )

Ejemplo: input.form-control.$@-*3
<input type="text" class="form-control 3"> <input type="text" class="form-control 2"> <input type="text" class="form-control 1">

8.2 Base numeros consecutivos ( @n )

Ejemplo: input.form-control.$@10*3
<input type="text" class="form-control 10"> <input type="text" class="form-control 11"> <input type="text" class="form-control 12">

9. subir de nivel ( ^ )

Ejemplo: div>label.radio^div>label.radio
<div><label for="" class="radio"></label></div> <div><label for="" class="radio"></label></div>

10. Agrupar elementos "( )"

Ejemplo: (header>ul.nav>li*5)+footer
<header> <ul class="nav"> <li></li> <li></li> <li></li> <li></li> <li></li> </ul> </header> <footer></footer>

11. Generador Lorem ipsum ( Lorem )

Ejemplo: lorem
Lorem ipsum dolor sit amet, consectetur adipisicing elit. Iste minima assumenda officiis sequi laudantium blanditiis sunt inventore expedita esse similique nam nobis, repellendus perferendis? Quis rerum eum sunt corrupti non.

12. Documentos HTML ( html )

Ejemplo: html:5
<!DOCTYPE html> <html lang="en"> <head> <meta charset="UTF-8"> <title>Document</title> </head> <body> </body> </html>
